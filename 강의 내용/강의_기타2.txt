for -> 배열 객체
for ~ in -> 객체의 속성과 값을 순회

++++++++++++++++++++++++++++++++++++++++++++++

in 연산자 -> 객체의 속성명이 존재하는지 체크
          -> 존재하면 true, 그렇지 않으면 false

          예) 'age' in person -> person 객체에 age 속성이 정의 되어 있는지..


JSON - JavaScript Object Notation - 자바스크립트 객체 표기법
{
  속성:값,
  속성:값,
}

this - 객체(인스턴스)의 주소를 내부에서 접근하는 변수
     - 객체(인스턴스) 내부의 속성에 접근할때

외부에선 person.age(); 이런식으로 접근
내부에선 this.age                         = 주소가 같다
//this와 preson은 같은 객체(인스턴스) 주소를 가지고 있다
차이점은 this는 객체 내부에서만 접근이 가능
